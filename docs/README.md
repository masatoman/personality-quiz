## 🔄 **ドキュメント管理ルール**

### ✅ **更新原則**
- **即座更新**: 仕様変更・決定事項は即座に反映
- **作業記録**: 重要な作業は`03_進捗管理/03_日誌/`に記録
- **テンプレート活用**: 新規ドキュメントは`99_テンプレート/`を使用

### 🔍 **品質基準**
- **明確性**: 誰が読んでも理解できる
- **実用性**: 実際の作業で使える
- **最新性**: 常に現在の状況を反映
- **網羅性**: 必要な情報が漏れなく記載

### 📱 **緊急時の使い方**
1. **問題発生** → [緊急時対応チェックリスト](./02_開発ガイド/05_トラブルシューティング/16_緊急時対応チェックリスト.md)
2. **デプロイ前** → [環境確認チェックリスト](./02_開発ガイド/03_デプロイ・運用/17_環境別確認チェックリスト.md)
3. **開発再開** → [再開時のチェックリスト](./03_進捗管理/05_再開時のチェックリスト.md)
